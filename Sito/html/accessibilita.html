Validità del codice

I diversi file che compongono questo sito web risultano validi per quanto riguarda il codice HTML 5 e quello CSS3. In tal modo, può garantire un livello minimo di accessibilità agli utenti che necessitano di tecnologie assistive. Tali file sono stati testati utilizzando i validatori ufficiali del consorzio W3C.

  • Validatore HTML: http://validator.w3.org/check
  • Validatore CSS3: http://jigsaw.w3.org/css-validator/

Accelleratori

Il sito web dispone di accelleratori da tastiera per offrire una navigazione completa utilizzando solamente tale periferica. La lettera corrispondente all'accelleratore viene indicata nel punto in cui può essere utilizzato.
